# Servicios de Data Science (Production ML) — Daniel Varela Perez

## Enfoque
Construyo soluciones end-to-end de ML listas para produccion: preparacion de datos, modelado, validacion, despliegue (API/batch) y monitoreo. Especialidad en fraude, riesgo crediticio y forecasting.

## Paquetes (entrada rapida + expansion)
### 1) ML Quickstart (2 semanas)
**Ideal para:** equipos que necesitan un primer modelo util y un demo ejecutivo.
**Incluye:**
- Pipeline de datos + baseline
- Entrenamiento y evaluacion
- Dashboard ejecutivo (Streamlit)
- Reporte de metricas y recomendaciones

### 2) Production Scoring API (3–4 semanas)
**Ideal para:** equipos con modelo listo pero sin despliegue.
**Incluye:**
- API FastAPI para scoring
- Batch scoring
- Validaciones basicas + monitoreo inicial
- Documentacion tecnica y runbook corto

### 3) Risk/Fraud Model Audit (1 semana)
**Ideal para:** equipos que ya usan modelos y necesitan mejoras.
**Incluye:**
- Evaluacion de performance
- Drift basico y analisis de estabilidad
- Plan de mejora priorizado

## Como trabajamos
1. Diagnostico rapido (30–45 min)
2. Definicion de alcance y entregables
3. Desarrollo con checkpoints semanales
4. Entrega + handoff (documentacion y pasos siguientes)
